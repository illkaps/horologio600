# Horologio600

## Running the App Locally

To run the React app locally, you need to have Node.js and npm installed on your machine. Once you've done that:

1. Clone the repository:
   ```bash
   git clone https://github.com/illkaps/horologio600.git
   ```

2. Navigate to the project directory:
   ```bash
   cd horologio600
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

This will start the React app and it will be available at `http://localhost:3000`.

## Deploying the App

To deploy the application, run the following command:
```bash
npm run deploy
```

Make sure you have set up the correct deployment configuration in your project before running this command.
